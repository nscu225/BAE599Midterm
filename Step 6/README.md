Step 6 - Interactive visualization

This folder contains an interactive Plotly HTML that visualizes U.S. rice yield over time with an overlayed linear trend.

Files:
- create_interactive.py : script to generate `interactive_yield.html` from `Step 2/cleaned_yield_data.csv`.
- interactive_yield.html : saved interactive figure (created when script runs).
- requirements.txt : dependencies.

How to run:

python3 "Step 6/create_interactive.py"

Open `Step 6/interactive_yield.html` in a browser to interact (hover, zoom, pan).
